# Funny Romantic Date Website 💌

A very small mobile-friendly static website to ask your fiancée for a date tomorrow.

## Files

- `index.html`
- `style.css`
- `script.js`

## Deploy on Vercel

1. Create a new GitHub repository.
2. Upload these files to the repository.
3. Go to Vercel and import the repository.
4. Framework preset: **Other**.
5. Build command: leave empty.
6. Output directory: leave empty or use `/`.

## Deploy on GitHub Pages

1. Create a GitHub repository.
2. Upload these files.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` branch and `/root`.
6. Save.

## Customize

Open `index.html` and edit:
- The question in the `<h1>`
- The subtitle text
- The sweet message inside `<div class="message">`
